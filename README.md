# 訪問作業記録アプリ（かじきろく）

訪問先（星さん宅）での家事お手伝いの内容を記録する、iPhone用の日報アプリです。

**目的**: 作業内容・物の移動や処分・現金や貴重品への接触の有無を客観的に記録し、あとから「物がなくなった」などの疑いをかけられたときに、自分を守る証拠を残すこと。

## 大事な特徴

- **記録・写真・音声はすべてあなたのiPhoneの中だけに保存されます。** インターネット上のサーバーには一切送信されません。
- 記録の**作成日時・修正日時は自動で記録**され、あとから変えられません。修正すると修正前の内容が履歴として残ります。
- 証拠としての信頼性を保つため、**記録を削除する機能はわざと付けていません**。
- インターネットがつながらない場所でも動きます（一度開けばオフラインで使えます）。

## iPhoneへの入れ方（最初に1回だけ）

1. iPhoneのSafariでアプリのURLを開く
2. 画面下の「共有」ボタン（四角に上矢印）をタップ
3. 「**ホーム画面に追加**」をタップ
4. ホーム画面にアイコンが追加されます。次からはこのアイコンから開いてください

> ⚠️ 必ず「ホーム画面に追加」をしてください。Safariで開くだけの状態だと、iPhoneの空き容量が減ったときにデータが自動削除されることがあります。

## 使い方

### 訪問したら

1. アプリを開いて「**＋ 新しい日報を作る**」をタップ（訪問開始日時は自動で入ります）
2. **作業前の写真**を撮る（そのままカメラが起動できます）
3. 作業をする。文字入力はキーボードの**マイクボタン**で音声入力すると楽です
4. 物を動かした・処分した場合は、その欄に「何を・どこから・どこへ」を書く
5. **貴重品チェック**：現金・通帳・印鑑・カードなどに触れたかを選ぶ（触れていなければそのまま）
6. **作業後の写真**を撮る
7. 帰る前に「🎤 終了時確認を録音」で、星さんとの確認のやりとりを短く録音
   （例：「本日の作業は以上です。荷物の確認をお願いします」→星さんの返事まで）
8. 「**保存する**」をタップ

### 帰宅後

- 記録を開いて内容を確認し、必要なら「✏️ 修正する」で追記（修正履歴が自動で残ります）
- 「💬 LINE報告文を作る」→「共有する」で、紹介者（相模さん）にそのまま送れる報告文が作れます
- 「🖨 PDF出力（印刷）」→ 共有メニューから「**PDFとして保存**」または「プリント」でPDF化できます

### バックアップ（週1回おすすめ）

1. ホーム右上の ⚙️ →「バックアップを書き出す」
2. パスワードを決めて入力（**紙にメモして自宅で保管**。忘れると復元できません）
3. 「🔒 暗号化して書き出す」→ 出てきたファイルを「ファイルに保存」でiCloud Driveへ

書き出したファイルは暗号化されているので、iCloudに置いても中身は誰にも読めません。
機種変更やデータが消えたときは、⚙️ →「バックアップから復元する」で戻せます。

## パソコンでの動かし方（開発・確認用）

このフォルダで次のコマンドを実行し、ブラウザで http://localhost:8000 を開きます。

```
py -m http.server 8000
```

※ アプリ本体はGitHub Pagesでホスティングされていますが、ページが配信するのは画面のプログラムだけで、あなたの記録データがサーバーに送られることはありません。

## 注意点

- **iPhoneの「設定 → Safari → 履歴とWebサイトデータを消去」を実行すると記録が全部消えます。** 絶対に実行しないでください（実行が必要なときは先にバックアップを）。
- ホーム画面からアプリを削除しても記録が消えます。
- 録音はiPhoneのマイク許可が必要です。初回に「許可」を選んでください。
- LINEへの送信は「共有シート」経由です（共有する → LINE → 送り先を選ぶ）。
- 記録番号・作成日時・修正日時はアプリが自動で付けるもので、手で書き換えることはできません。
